Introducción

Esta página encontraras una forma sencilla para programar en PHP utilizando la arquitectura de software MVC.

Surgue esta documentación a partir de las necesidades de que en internet se encuentra mucha documentación relaciada pero nada en especifico, basandose en modelos mal hechos y estructuras mal programadas.

Ventajas:
PDO nos permite una abstracción para el acceso a datos. Es decir que las funciones para realizar consultas y para obtener datos son las mismas en cualquier servidor SQL utilizado (MySQL, PostgreSQL, ...). Por ejemplo, si pasas de MySQL a PostegreSQL, no necesitas transformar todo tu código como con las funciones mysql_ (mysql_connect sería reemplazado por pgsql_connect, mysql_query por pgsql_query, ...). PDO es también orientado a objetos, por lo que podrás extenderlo y administrar los errores con las excepciones.
